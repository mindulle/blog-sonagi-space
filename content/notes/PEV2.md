# PEV2

`pev2`, or *Postgres Explain Visualizer v2*, is an open-source tool designed to make query analysis with PostgreSQL easier and more understandable. By providing a visual representation of the `EXPLAIN ANALYZE` output, `pev2` simplifies query optimization by displaying the query plan and execution me
trics in a readable structure.

Learn more from the following resources:

- [@opensource@dalibo/pev2](https://github.com/dalibo/pev2)
