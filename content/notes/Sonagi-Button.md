# Sonagi Button

소나기 디자인 시스템의 가장 기본적인 상호작용 컴포넌트입니다. **'Wild & Thin'** 아이덴티티를 기반으로, 견고한 선과 명확한 대비감을 강조합니다.

## 1. 디자인 컨셉
- **Wild**: `outline` 변형 시 로고의 획 굵기와 일치하는 2px 테두리(`--border-base`)를 사용하여 강한 인상을 줍니다.
- **Thin**: 기본 상태에서는 절제된 여백과 정교한 타이포그래피를 유지합니다.

## 2. 기술적 명세 (Implementation)
shadcn/ui의 구조를 계승하며, `variables.css`의 토큰을 직접 참조합니다.

### 코드 구조 (React/Tailwind)
```tsx
const buttonVariants = cva(
  "inline-flex items-center justify-center transition-all active:scale-95 disabled:opacity-50",
  {
    variants: {
      variant: {
        primary: "bg-[var(--color-brand-primary)] text-[var(--color-text-inverse)] hover:opacity-90",
        outline: "border-[var(--border-base)] border-[var(--color-text-primary)] text-[var(--color-text-primary)] bg-transparent hover:bg-[var(--color-bg-overlay)]",
        ghost: "text-[var(--color-text-primary)] hover:bg-[var(--color-bg-overlay)]",
      },
      size: {
        default: "h-10 px-6 py-2 rounded-[var(--radius)]",
        sm: "h-8 px-3 rounded-[var(--radius)] text-xs",
        lg: "h-12 px-10 rounded-[var(--radius)] font-[var(--font-serif)] text-lg",
      },
    },
    defaultVariants: {
      variant: "primary",
      size: "default",
    },
  }
)
```

## 3. 토큰 매핑 확인
| 속성 | 사용된 토큰 | 효과 |
| :--- | :--- | :--- |
| 배경색 | `--primary` | 소나기 블루 (`#1991b9`) |
| 테두리 | `--border-base` | 2px의 두꺼운 실선 (Outline 시) |
| 서체 (기본) | `--font-sans` | Pretandard (가독성 중심) |
| 서체 (Lg) | `--font-serif` | Classic Serif (장식적 강조) |

## 4. 사용 가이드
- **언제 사용하는가?**: 주요 액션(저장, 제출)에는 `primary`, 보조 액션(취소, 목록)에는 `outline` 또는 `ghost`를 사용합니다.
- **주의사항**: `lg` 사이즈 버튼은 문서의 헤드라인과 결합될 때 최적의 시각적 조화를 이룹니다.
