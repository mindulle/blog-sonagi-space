# 🛡️ 생활비 방어 플랜 (Stabilization Plan)

이 문서는 다른 전략을 실행하기 전, **최소한의 월 생활비를 100% 확보**하기 위한 정기적이고 반복적인 '노동 엔진' 구축을 목표로 합니다.

## 1. 월 목표 수익 설정 (Target Calculation)
*사용자님의 상황에 맞춰 아래 금액을 수정해 주세요.*

| 항목 | 추정 금액 (원) | 비고 |
| :--- | :--- | :--- |
| 식비 및 생필품 | 600,000 | |
| 주거비 (월세/관리비) | 500,000 | |
| 통신 및 교통비 | 150,000 | |
| 여유 자금 및 기타 | 250,000 | |
| **최종 목표액** | **약 1,500,000** | **하루 7.5만원 확보 목표 (주 5일 기준)** |

## 2. '노동 엔진' 플랫폼 선정 (Workhorse Platforms)
일감이 끊기지 않고 지속 가능한 플랫폼 2개를 주력으로 삼습니다.

### 엔진 1: DataAnnotation.tech (Global High-Volume)
- **목표 수익**: 시간당 $20 (약 2.7만원)
- **필요 시간**: 하루 3시간 집중 노동 = **하루 8.1만원**
- **지속성**: 2026년 기준, 퀄리티 유지만 된다면 일감이 상시 존재.

### 엔진 2: Crowdworks (Local Backup)
- **목표 수익**: 일당 2~3만원 (짜투리 시간 활용)
- **필요 시간**: 이동 시간이나 대기 시간 활용.
- **지속성**: 한국어 기반이라 심리적 피로도가 낮음.

## 3. 데일리 루틴 (Daily Minimum Goal - DMG)
"무조건 이만큼은 채우고 잔다"는 원칙입니다.

- **09:00 ~ 12:00**: **DataAnnotation** 집중 세션 (목표 $60 확보)
- **14:00 ~ 15:00**: **UserTesting** 대기 및 소액 태스크 (목표 $10 확보)
- **상시**: **모니모/토스** 퀴즈 및 **크라우드웍스** 모바일 작업 (목표 0.5만원)
- **[합계]**: **일일 약 10만원** (환율 1,350원 기준) -> **월 200만원 (주말 휴식)**

## 4. 에이전트 자동화 지원 (Agent Lever)
지속 가능성을 위해 '노동의 피로도'를 인공지능이 덜어줍니다.

1.  **마루 (Librarian)**: 
    - 3시간 연속 작업 시 집중력이 떨어지지 않게 '검수 요약본' 제공.
    - 플랫폼 정책 변경 사항 상시 모니터링하여 계정 정지 리스크 관리.
2.  **제니 (Designer)**: 
    - 작업 환경(책상, 조명 등) 가이드를 제공하여 장시간 PC 작업 시의 건강 관리.
3.  **머니 (Money)**: 
    - 매일 저녁 '오늘의 달러 수입'을 한화로 환산하여 대시보드 업데이트.
