# 📚 Study: Zelda BOTW UI Kit (Sheikah Slate Style)

이 문서는 세계적인 게임 디자인의 정수로 꼽히는 **젤다의 전설: 브레스 오브 더 와일드(BOTW)**의 UI 시스템을 분석한 리포트입니다. 소나기(Sonagi) 프로젝트에 **게임화(Gamification)** 요소를 도입하거나, 기술 위키의 딱딱함을 깨는 '몰입형 테마'를 설계하기 위한 레퍼런스로 작성되었습니다.

## 2. 💡 Key Insights for Sonagi v3.0

### 2.1 Immersive Information Layering
- **분석**: 게임의 몰입을 방해하지 않으면서도 수많은 상태 정보(체력, 스태미나, 온도, 소음 등)를 화면 구석에 효율적으로 배치합니다.
- **소나기 v3.0 적용**: 위키의 대시보드 상단에 사용자의 **'지식 축적 스태미나'**나 **'위키 건강도(Health)'**를 게임 UI 스타일의 인디케이터로 시각화하여, 지식 관리를 하나의 퀘스트처럼 느끼게 만듭니다.

### 2.2 Functional Ornaments
- **분석**: 텍스트 박스의 모서리 장식(`Text Ornament Corner`)이나 타이틀 장식들이 단순히 미적 요소를 넘어, 정보의 시작과 끝을 명확히 구분하는 경계선 역할을 합니다.
- **적용**: 소나기의 'Wild' 레이어에서 2px 보더를 보완하는 요소로, 이러한 **기하학적 오너먼트**를 도입하여 시스템의 개성을 부여합니다.

### 2.3 Semantic Iconography
- **Icons**: 퀘스트 타입(Main, Side, Shrine)에 따라 아이콘의 형태와 컬러를 엄격히 구분합니다.
- **적용**: 소나기 위키의 문서 타입(`_concepts`, `_resources`, `_people`)별 아이콘 설계 시, BOTW의 **상징 중심의 직관적 아이콘 체계**를 벤치마킹합니다.
