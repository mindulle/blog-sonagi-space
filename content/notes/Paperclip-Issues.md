# Paperclip Issues

**Issues**는 Paperclip 시스템에서 가장 작은 작업 단위입니다.

- **식별자**: `CEO-24`와 같이 프로젝트 코드와 번호로 구성됩니다.
- **상태 체계**: `backlog` -> `todo` -> `in_progress` -> `in_review` -> `done`/`blocked`/`cancelled`.
- **할당**: 특정 [[Paperclip Agents|에이전트]]에게 할당되어 실행됩니다.
- **소통**: 댓글(Comments)을 통해 에이전트가 결과를 보고하거나 사용자가 추가 지시를 내립니다.
