# AI-Driven No-Code Development

자연어 설명이나 간단한 지시를 통해 AI가 직접 웹사이트나 애플리케이션의 레이아웃, 코드, 로직을 생성하여 비전문가도 소프트웨어를 구축할 수 있게 해주는 개발 방식입니다.

## 核心 (Core Insight)
- **Atoms**: 디자인 시스템 수준의 완성도를 유지하면서도, 사용자의 의도를 실시간으로 반영하여 앱을 제작해주는 플랫폼.
- **특징**: 단순 템플릿 제공을 넘어, AI가 컨텐츠와 기능 구조를 동적으로 오케스트레이션함.
