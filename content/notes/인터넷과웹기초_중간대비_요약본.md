# 🌐 인터넷과웹기초 중간고사 단권화 노트

> 본 문서는 수업 강의 자료(HTML5, CSS3, JavaScript 기초)를 기반으로 작성된 중간고사 대비 요약본입니다.

## 📌 1. 인터넷과 웹의 이해
* **인터넷의 탄생**: 1969년 미국 국방성 산하 ARPA가 구축한 ARPANET이 시초. TCP/IP 프로토콜 기반.
* **WWW (World Wide Web)**: 팀 버너스 리 제안. 하이퍼텍스트 문서를 HTTP로 전달.
* **웹 브라우저 전쟁**:
  * 1차: 넷스케이프 vs IE (ActiveX, Flash 의존 심화, 비표준 문제)
  * 2차: 구글 크롬(V8 엔진) 등장 및 웹 표준의 중요성 부각
  * 현대: 구글 크롬(블링크/V8), 사파리(웹킷), 파이어폭스(게코/스파이더몽키)
* **웹 표준 기구**:
  * **W3C**: 웹 표준 제정 (CSS WG 등)
  * **WHATWG**: 버전 없는 Living Standard로 HTML/DOM 지속 개선 (현재 HTML 표준 단일 출판자)

## 📌 2. HTML5 기본 구조와 시맨틱 태그
* **HTML (HyperText Markup Language)**: 문서의 구조와 의미(시맨틱)를 정의.
* **기본 구조**:
  ```html
  <!DOCTYPE html> <!-- HTML Living Standard 준수 선언 -->
  <html lang="ko">
  <head> <!-- 메타 정보: charset, title, 외부 css/js 링크 --> </head>
  <body> <!-- 실제 브라우저에 렌더링되는 콘텐츠 --> </body>
  </html>
  ```
* **시맨틱 태그 (Semantic Tags)**: 문서의 의미 구조를 명확히 표현하여 SEO 및 웹 접근성 향상.
  * `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>`
* **주요 태그**:
  * **텍스트**: `h1`~`h6`, `p`, `a` (절대/상대/아이디 경로 지원)
  * **목록**: `ul` (순서 없음), `ol` (순서 있음), `li`
  * **미디어**: `img` (alt 속성 필수), `audio`, `video`
  * **테이블**: `table`, `tr`, `th`, `td` (셀 병합: `colspan`, `rowspan`)
  * **폼(Form)**: `form`, `fieldset`, `legend`, `input` (text, password, radio, checkbox, file 등), `select`, `textarea` (GET/POST 전송 방식 구분 중요)
  * **비시맨틱**: `div` (블록 레벨 컨테이너), `span` (인라인 컨테이너)

## 📌 3. CSS3 핵심 개념
* **CSS (Cascading Style Sheets)**: 웹 문서의 시각적 표현(스타일, 레이아웃) 담당.
* **선택자 (Selectors)**:
  * 전체(`*`), 타입(`태그`), 아이디(`#id`), 클래스(`.class`)
  * **결합자**: 후손(`A B`), 직계 자식(`A > B`), 형제(`A + B`, `A ~ B`)
  * **가상 클래스 (상태/반응)**: `:hover`, `:active`, `:focus`, `:checked`, `:disabled`
  * **구조 선택자**: `:first-child`, `:last-child`, `:nth-child(n)`
  * **속성 선택자**: `[type="text"]`, `a[target="_blank"]`
* **크기 단위**: `px`, `%`, `em` (부모 요소 폰트 기준), `rem` (최상위 html 폰트 기준), `vw/vh`
* **박스 모델 (Box Model)**:
  * 구성 요소: `content`, `padding`, `border`, `margin`
  * 마진 겹침(Margin Collapse): 인접한 블록 요소의 상하 마진은 더 큰 값으로 겹침.
  * **`box-sizing`**: `content-box`(기본) vs `border-box`(실무 필수, 너비에 테두리/패딩 포함)
* **`display` 속성**:
  * `block`: 한 줄 전체 차지, width/height 적용 O
  * `inline`: 글자처럼 배치, width/height 적용 X
  * `inline-block`: 나란히 배치되면서 width/height 적용 O
  * `none` (공간 차지 X) vs `visibility: hidden` (공간 차지 O)
* **위치 지정 (`position`)**:
  * `static`: 기본 흐름
  * `relative`: 원래 위치 기준 이동
  * `absolute`: 가장 가까운 위치 지정된 조상 기준 배치 (문서 흐름에서 빠짐)
  * `fixed`: 브라우저 뷰포트 기준 고정 (스크롤 무시)
* **레이아웃 (`Flexbox`)**: 1차원 행/열 배치
  * `flex-direction`: 주축 설정 (row, column)
  * `justify-content`: 주축 정렬 (flex-start, center, space-between, space-evenly 등)
  * `align-items`: 교차축 정렬 (stretch, center, baseline 등)
  * `flex-wrap`: 아이템 줄바꿈 처리
  * `flex-grow`, `flex-shrink`, `flex-basis`: 여유 공간 분배 및 축소, 기본 크기 설정

## 📌 4. JavaScript 기초 및 DOM
* **역할**: 웹 페이지의 동적 기능, 이벤트 처리, 서버 비동기 통신.
* **DOM (Document Object Model)**: 문서를 노드(Node) 객체의 트리 구조로 표현하는 프로그래밍 인터페이스.
  * JavaScript와 DOM은 분리된 개념. JS가 DOM API를 통해 조작.
* **HTML 주요 전역 객체**:
  * `document`: 문서 전체 접근 (진입점)
  * `window`: 브라우저 창 전체 (최상위 객체)
  * `location` (주소), `history` (방문기록), `navigator` (환경 정보), `screen` (화면 정보)
* **핵심 DOM 제어 API**:
  * `document.querySelector()`: 요소 검색
  * `document.createElement()`: 요소 생성
  * `Element.innerHTML`: 마크업 읽기/할당
  * `EventTarget.addEventListener()`: 이벤트(click 등) 콜백 등록
