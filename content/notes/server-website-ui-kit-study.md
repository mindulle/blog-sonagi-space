# 📚 Study: Server Website UI Kits

## 2. 💡 Key Insights for Sonagi v3.0

### 2.1 Infrastructure Visualization
- **분석**: `vuesax/bold/cloud`와 같은 볼드한 아이콘 세트를 사용하여 서비스의 핵심 가치를 강조합니다.
- **소나기 v3.0 적용**: 소나기의 'Infrastructure' 위키 페이지에서 현재 서버의 상태(CPU, Memory, Network)를 시각화할 때, 이 킷의 **볼드하고 명확한 상징 체계**를 도입하여 가독성을 높입니다.

### 2.2 Data-centric Card Layouts
- **분석**: 복잡한 사양(Specs)과 가격 정책을 비교하기 쉬운 카드 레이아웃으로 정리하는 방식이 돋보입니다.
- **적용**: 소나기 v3.0의 'Projects' 대시보드에서 각 개발 프로젝트의 기술 스택과 서버 환경을 요약해서 보여줄 때 이 카드 구조를 벤치마킹합니다.

### 2.3 Abstract Backgrounds for Technical Context
- **적용**: 기술적이고 딱딱해 보일 수 있는 내용을 부드럽게 완화하기 위해, 이 킷에서 사용된 **추상적인 그라데이션과 텍스처**를 소나기의 'Wild' 레이어 배경 요소로 부분적으로 채용하는 것을 고려합니다.
