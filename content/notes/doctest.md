# Doctest

Python’s standard library comes equipped with a test framework module called doctest. The doctest module programmatically searches Python code for pieces of text within comments that look like interactive Python sessions. Then, the module executes those sessions to confirm that the code referenced b
y a doctest runs as expected.

Visit the following resources to learn more:

- [@official@Doctest Module](https://docs.python.org/3/library/doctest.html)
- [@article@How To Write Doctests in Python](https://www.digitalocean.com/community/tutorials/how-to-write-doctests-in-python)
