### 📱 1. 반응형 웹 & CSS 단위
> 💡 **가변 단위 완벽 비교**
> * `vw` / `vh`: 뷰포트 너비/높이의 1%.
> * `em`: 부모 폰트 기준 $\rightarrow$ **중첩 시 크기 폭발 주의**.
> * `rem`: **최상위(html) 폰트 기준** $\rightarrow$ 중첩 문제가 없어 가장 안전함.

* **뷰포트 필수 태그**: `<meta name="viewport" content="width=device-width, initial-scale=1.0">`
* **미디어 쿼리 (Media Query)**:
  * `@media screen`: 프린트 매체 무시, 화면(모니터/폰)에만 적용.
  * **데스크탑 퍼스트**: PC(기본) $\rightarrow$ 모바일 (`max-width`로 좁힘)
  * **모바일 퍼스트**: 모바일(기본) $\rightarrow$ PC (`min-width`로 넓힘)

### ⚡ 2. 모던 JS 문법 & 호이스팅
> 💡 **호이스팅과 TDZ (함정 주의)**
> `let`과 `const`도 끌어올려지지만, 선언 전에 접근 시 **TDZ(일시적 사각지대)**에 빠져 에러(`ReferenceError`) 발생.

* **변수 선언 차이점**:
  * `var`: 함수 스코프, 재선언 허용 (구형).
  * `let`: 블록 스코프 `{}`, 재할당 가능.
  * `const`: 블록 스코프 `{}`, 재할당 불가. 단, **배열/객체 내부 요소는 수정 가능**.
* **동등 vs 일치 연산자**:
  * `==`: 값만 대충 비교 (`1 == '1'` $\rightarrow$ `true`)
  * `===`: **값 + 자료형** 엄격 비교 (`1 === '1'` $\rightarrow$ `false`)
* **Falsy (거짓 평가 값)**:
  * 오직 6개: `0`, `""`, `null`, `undefined`, `NaN`, `false` 
  * (주의: 빈 문자열이 아닌 `"false"`, `[]`, `{}`는 모두 *Truthy*임)

### 🔄 3. 배열 제어 & 화살표 함수
> 💡 **체이닝 활용 팁**
> 배열 원본 변경 없이 **.filter(조건).map(변형)** 순서로 엮어서 새 배열을 만들어내는 패턴 출제 빈도 높음!

* **배열 원본 변경 메서드**: `push`(뒤 추가), `pop`(뒤 삭제)
* **새 배열 반환 메서드**: `map`(일괄 변형), `filter`(조건 추림)
* **화살표 함수 (Arrow Function)**:
  * 문법: `const add = (a, b) => a + b;`
  * 인자가 1개면 괄호 생략 가능: `x => x * 2`
  * 본문 중괄호 `{}`를 생략하면 자동 `return` 됨.
  * 단, **객체 `{id: 1}`을 즉시 반환하려면 `({id: 1})` 소괄호 래핑 필수**.

### 🖱️ 4. DOM 제어와 이벤트 심화
> 💡 **이벤트 제어 (★출제 1순위)**
> * `e.preventDefault()`: 링크 이동, 폼 제출 등 **브라우저 기본 동작 차단**.
> * `e.stopPropagation()`: 상위로 올라가는 **버블링(Bubbling) 차단**.

* **텍스트 노출 속성 차이**:
  * `innerHTML`: 태그 파싱. (보안 취약)
  * `innerText` / `textContent`: 태그 무시, **오직 텍스트만** 삽입.
* **이벤트 위임 (Event Delegation)**:
  * JS로 동적 추가된 자식 요소는 미리 리스너를 달 수 없음.
  * 부모에 리스너를 달고 `e.target`(실제 클릭된 자식)과 `e.currentTarget`(이벤트 달린 부모) 구분.

### 🧰 5. 실전 jQuery 라이브러리
> 💡 **속성 제어 (attr vs prop)**
> HTML에 적힌 문자열 그대로는 `.attr()`을 쓰지만, 체크박스의 `checked` 처럼 **동적으로 변하는 상태값**은 무조건 `.prop()`으로 제어할 것!

* **계층 탐색 핵심**:
  * `.find('.item')`: 자손 요소. (띄어쓰기 `$('#box .item')`과 동일)
  * `.closest('.card')`: 나부터 시작해 조상으로 거슬러 올라가 **가장 먼저 만나는 부모** 탐색. (이벤트 위임용)
* **순수 JS $\rightarrow$ jQuery 변환 규칙**:
  * `el.style.display = 'none'` $\rightarrow$ `$(el).hide()`
  * `el.className += ' active'` $\rightarrow$ `$(el).addClass('active')`
  * `DOMContentLoaded` $\rightarrow$ `$(document).ready(...)`

### 🌐 6. 비동기 통신 & 데이터 (Fetch)
> 💡 **Fetch API & async/await**
> `await` 키워드는 **반드시 `async` 함수 내부에서만** 사용 가능. 예외 상황 대처를 위해 `try ... catch` 블록으로 감싸는 것이 권장됨.

* **JSON (JavaScript Object Notation)**:
  * **엄격한 문법**: 키(Key)와 문자열 값은 *무조건 큰따옴표("")* 사용!
  * `JSON.stringify(obj)`: JS 객체 $\rightarrow$ 문자열 (네트워크 전송용)
  * `JSON.parse(str)`: 문자열 $\rightarrow$ JS 객체로 파싱
* **프로미스 (Promise) 상태**:
  * `Pending`(대기), `Fulfilled`(이행 $\rightarrow$ `.then()`), `Rejected`(실패 $\rightarrow$ `.catch()`)
