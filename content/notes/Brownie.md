# Brownie

Brownie is a Python-based development and testing framework for smart contracts targeting the Ethereum Virtual Machine (EVM). It simplifies the process of writing, deploying, testing, and interacting with smart contracts by providing a suite of tools and libraries tailored for Python developers. Bro
wnie aims to make smart contract development more accessible and efficient, especially for those already familiar with Python.
