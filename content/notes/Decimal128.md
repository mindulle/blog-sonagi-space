# Decimal128

`Decimal128` data type in MongoDB provides exact decimal representation for financial and monetary calculations requiring precision. Based on IEEE 754-2008 standard, it supports 34 decimal digits with exact arithmetic operations. `Decimal128` eliminates floating-point precision errors, making it ess
ential for applications handling currency, accounting, and scientific computations where decimal accuracy is critical.
