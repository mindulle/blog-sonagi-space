# fmt

`fmt` is a modern formatting library for C++. It provides a simple and efficient way to format text, similar to Python's `str.format()` or C#'s `string.Format()`. It offers compile-time checks for format string correctness and generally results in faster and safer string formatting compared to tradi
tional methods like `printf`.

Visit the following resources to learn more:

- [@official@fmt](https://fmt.dev/12.0/)
- [@opensource@fmt](https://github.com/fmtlib/fmt)
