# Lazy loading

Lazy loading is a technique in Angular that allows you to load JavaScript components asynchronously when a specific route is activated. It improves the application load time speed by splitting the application into several bundles. The bundles are loaded as required when the user navigates through th
e app.

Visit the following resources to learn more:

- [@official@Lazy-loading Feature Modules](https://angular.dev/guide/ngmodules/lazy-loading)
- [@video@Angular Tutorial - Lazy Loading](https://www.youtube.com/watch?v=JjIQq9lh-Bw)
