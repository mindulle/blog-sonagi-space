# Bulk Operations

Cloudflare Workers allow you to perform bulk operations on KV stores. You can use the `batch` method to perform multiple operations in a single request. This is useful for tasks like importing data from an external source or updating multiple records at once. Bulk operations are efficient and scalab
le, allowing you to handle large amounts of data with ease.

Visit the following resources to learn more:

- [@official@Workers KV](https://blog.cloudflare.com/workers-kv-is-ga/)
- [@official@Workers KV Runtime API](https://developers.cloudflare.com/workers/runtime-apis/kv)
