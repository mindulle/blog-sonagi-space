# Model Context Protocol (MCP)

## 요약
LLM과 데이터 소스 간의 연결을 표준화하여 풍부한 문맥을 안전하게 제공하고, 에이전트 간 문맥을 공유하는 프로토콜입니다.

## 상세 내용
MCP는 에이전트가 LLM에게 프롬프트를 보내기 전 배경 정보를 어떻게 구성할지 정의하는 표준 규칙입니다. 또한, AI 에이전트가 환경, 사용자, 목표에 대한 이해를 표준화된 방식으로 관리하고 공유할 수 있게 하여, 시스템 구성 요소 간 또는 다중 에이전트 간의 효율적인 협업을 가능하게 합니다. 일관된 형식의 데이터 전송을 통해 모델 교체가 용이하고 테스트 효율성이 높아집니다.
