# 💸 안정적 소액 수익 모델 (Micro-jobs)

이 페이지는 특별한 자격이나 큰 자본 없이, **노력한 만큼 확실하게 보상을 받을 수 있는** 온라인 소액 수익 모델들을 정리합니다. 수익은 상대적으로 적을 수 있으나, 플랫폼의 신뢰도가 높아 **미지급 리스크가 거의 없는** 것들 위주로 선별했습니다.

## 1. AI 데이터 라벨링 및 학습 (High Demand)
AI 모델(GPT-4, Claude 등)을 훈련시키기 위해 텍스트를 검수하거나 이미지를 분류하는 작업입니다.

| 플랫폼 | 주요 작업 | 특징 | 지급 방식 |
| :--- | :--- | :--- | :--- |
| **DataAnnotation.tech** | 챗봇 답변 평가, 코딩 검수 | **최고 단가 ($20/hr~)**, 일감이 꾸준함 | PayPal |
| **Outlier (Scale AI)** | RLHF, 팩트체크, 작문 | AI 선두 기업 Scale AI 운영, 작업량 많음 | PayPal |
| **Remotasks** | 이미지 태깅, LiDAR 데이터 처리 | 초보자 접근성 좋음, 난이도별 차등 지급 | PayPal |

## 2. 사용자 경험(UX) 및 웹/앱 테스트
웹사이트나 앱을 직접 써보고 느낀 점을 비디오/오디오로 피드백하는 작업입니다.

| 플랫폼 | 주요 작업 | 수익 | 특징 |
| :--- | :--- | :--- | :--- |
| **UserTesting** | 웹/앱 사용성 테스트 | $10 (20분 기준) | **업계 표준**, 매우 투명한 지급 체계 |
| **User Interviews** | 1:1 화상 인터뷰 | $50~$150 / hr | 특정 타겟에 맞을 경우 고수익 가능 |
| **uTest** | 소프트웨어 버그 찾기 | 버그당 차등 지급 | 실력이 쌓일수록 수익 급증 (Meritocracy) |

## 3. 마이크로 태스크 (Micro-tasks)
아주 간단한 작업(데이터 입력, 설문조사 등)을 수행하고 소액을 적립합니다.

| 플랫폼 | 주요 작업 | 특징 | 지급 방식 |
| :--- | :--- | :--- | :--- |
| **Prolific** | 학술 연구용 설문조사 | **거절 위험 낮음**, 연구자가 자금 선입금 | PayPal (즉시성) |
| **Amazon MTurk** | 데이터 분류, 텍스트 추출 | 전 세계 최대 규모, 일감이 끊이지 않음 | Amazon Pay / Bank |
| **Freecash / JumpTask** | 앱 설치, 게임 플레이 | 현금 회전 매우 빠름 (최소 출금액 낮음) | Crypto / PayPal |

## 🛡️ 안전 가이드: 돈을 떼먹히지 않으려면?
1.  **플랫폼 선정**: Trustpilot 평점 4.0 이상, 사용자 후기가 검증된 대형 플랫폼만 사용합니다.
2.  **직거래 금지**: 텔레그램이나 카카오톡으로 개별 연락하여 "먼저 보증금을 입금하라"는 곳은 100% 사기입니다. 반드시 위 플랫폼 내 시스템을 통해서만 정산받으세요.
3.  **다각화**: 특정 플랫폼의 일감이 끊길 수 있으므로(Empty Queue), 최소 3개 이상의 플랫폼에 계정을 만들어 두는 것이 유리합니다.
