# Core Graphics

Core Graphics, also known as Quartz, is a powerful 2D graphics rendering framework in iOS. It provides essential functions for drawing shapes, images, and text, handling low-level graphic operations with precision and efficiency. Core Graphics supports advanced features like color management, path-b
ased drawing, and anti-aliasing, enabling developers to create detailed and visually appealing graphics within their applications.

Learn more from the following resources:

- [@official@Core Graphics Documentation](https://developer.apple.com/documentation/coregraphics)
