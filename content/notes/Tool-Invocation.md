# Tool Invocation

## 요약
에이전트가 외부 도구를 선택하고 실행하여 현실 세계와 상호작용하는 단계입니다.

## 상세 내용
Acting 또는 도구 호출이라고도 하며, 에이전트가 세운 계획에 따라 적절한 도구(웹 검색, DB 쿼리, 계산기 등)를 선택하고 필요한 입력을 채워 넣는 과정입니다. 외부 시스템의 처리 결과를 받아 다음 행동을 결정하는 판단 근거로 사용합니다.
