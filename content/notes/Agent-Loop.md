# Agent Loop

## 요약
에이전트가 목표 달성을 위해 관찰, 판단, 행동을 반복하는 핵심 주기입니다.

## 상세 내용
에이전트 루프는 에이전트가 자율적으로 작동하게 하는 핵심 메커니즘입니다. 1) 신선한 데이터 수집, 2) 내부 상태 업데이트 및 추론/계획, 3) 도구 호출이나 API 요청 등의 행동 수행, 4) 결과 확인 및 메모리 저장의 단계를 반복하며 목표에 다가갑니다.
