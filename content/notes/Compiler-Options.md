# Compiler Options

Compiler options in TypeScript provide fine-grained control over how the TypeScript compiler (`tsc`) transforms `.ts` files into JavaScript. These options are typically configured within a `tsconfig.json` file or passed directly via the command line.  They dictate aspects like target ECMAScript vers
ion, module system, strictness of type checking, source map generation, and many other behaviors of the compilation process.

Visit the following resources to learn more:

- [@official@Compiler Options](https://www.typescriptlang.org/docs/handbook/compiler-options.html)
