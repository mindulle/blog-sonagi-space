# Adobe XD

Adobe XD (Experience Design) is a powerful design and prototyping tool that allows UX designers to create wireframes, mockups, and interactive prototypes for various digital projects. It is available for both Mac and Windows, and it focuses on providing an easy-to-use, intuitive interface for design
ing responsive websites, mobile apps, and more.

Visit the following resources to learn more:

- [@official@Adobe XD Platform](https://adobexdplatform.com/)
- [@official@Getting Started with Adobe XD](https://helpx.adobe.com/xd/get-started.html)
- [@official@Learn Adobe XD](https://www.adobe.com/ph_en/products/xd/learn/get-started-xd-design.html)
