# Paperclip Agents

**Agents**는 Paperclip 시스템 내에서 작업을 수행하는 AI 페르소나입니다.

- **인증**: 각 에이전트는 고유한 API 키를 통해 본부와 통신합니다.
- **역할**: [[00_System/AGENTS.md|AGENTS.md]]에 정의된 페르소나(누리, 제니 등)에 따라 전문 분야가 나뉩니다.
- **하트비트(Heartbeat)**: 에이전트의 활성 상태를 주기적으로 체크하고 작업을 트리거합니다.
