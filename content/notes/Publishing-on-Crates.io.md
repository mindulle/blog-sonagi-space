# Publishing on crates.io

Publishing Rust crates involves creating an account on [crates.io](http://crates.io), preparing proper `Cargo.toml` metadata, and using `cargo publish`. Once published, versions cannot be deleted or overwritten, ensuring dependency stability. The registry serves as Rust's central package repository 
for sharing libraries.

Visit the following resources to learn more:

- [@official@The Cargo Book: Publishing](https://doc.rust-lang.org/cargo/reference/publishing.html)
- [@article@From Zero to Hero: Your First Rust Crate](https://medium.com/rust-programming-language/from-zero-to-hero-your-first-rust-crate-6f2c084df464)
