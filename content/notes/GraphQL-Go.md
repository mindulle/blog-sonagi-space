# GraphQL Go

GraphQL Go refers to implementing GraphQL servers and clients using the Go programming language. Popular libraries include graphql-go/graphql for schema-first development and 99designs/gqlgen for code-first generation. Go's strong typing and performance make it excellent for building scalable GraphQ
L APIs.

Visit the following resources to learn more:

- [@opensource@graphql-go/graphql](https://github.com/graphql-go/graphql)
- [@opensource@99designs/gqlgen](https://github.com/99designs/gqlgen)
