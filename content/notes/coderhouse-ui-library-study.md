# 📚 Study: Coderhouse UI Library

이 문서는 라틴 아메리카의 선도적인 온라인 교육 플랫폼인 **Coderhouse**의 UI 라이브러리를 분석한 리포트입니다. 소나기(Sonagi) 프로젝트가 지향하는 '지식 공유 및 학습 커뮤니티'로서의 정체성을 강화하고, 카테고리별 시각적 분류 시스템을 구축하기 위한 레퍼런스로 작성되었습니다.

---

## 2. 💡 Key Insights for Sonagi v3.0

### 2.1 Category Color Coding (컬러 기반 분류)
- **분석**: 마케팅(Pink), 프로그래밍(Purple), 디자인(Blue) 등 각 교육 카테고리에 고유한 컬러를 부여하여 사용자가 수많은 강좌 중 원하는 분야를 즉각 인지하게 만듭니다.
- **소나기 v3.0 적용**: 위키의 대분류(`Computing`, `Design`, `Business`)별로 포인트 컬러를 지정하되, 'Wild' 레이어의 2px 보더 색상을 해당 카테고리 컬러로 가변 처리하여 지식의 소속을 시각화합니다.

### 2.2 Branding through Typography (아이덴티티 강화)
- **Titles**: `GOBOLD BOLD ITALIC`과 같은 파격적인 폰트를 제목에 사용하여 시스템의 에너지와 전문성을 동시에 표현합니다.
- **소나기 v3.0 적용**: 우리도 핵심 브랜드 키워드나 강조 섹션에 일반적인 본문 서체와 대비되는 **'Wild'한 타이포그래피**를 도입하여 지식의 지루함을 상쇄합니다.

### 2.3 Slant & Dynamic Patterns (역동적 패턴)
- **분석**: 사선(Slant) 형태의 레이아웃과 반복되는 텍스트 패턴을 통해 정적인 화면에 리듬감을 부여합니다.
- **적용**: 위키의 빈 페이지나 로딩 화면 등에 Coderhouse 스타일의 **텍스트 패턴 배경**을 적용하여 브랜드의 생동감을 유지합니다.
