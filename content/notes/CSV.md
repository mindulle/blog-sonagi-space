# CSV

CSV (Comma Separated Values) is a plain text file format used to store tabular data, such as spreadsheets or databases. Each line in a CSV file represents a row of data, and the values within each row are separated by commas. It's a simple and widely supported format for exchanging data between diff
erent applications.
