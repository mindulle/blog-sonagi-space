# Agent Proficiency

**Agent Proficiency (에이전트 활용 능력)**은 안드레 카파시(Andrej Karpathy)가 강조한 개념으로, AI 에이전트에게 적절한 지시(Prompt)를 내리고 도구(Tool)를 연결하여 복잡한 컴퓨터 작업을 자동화하는 능력을 의미합니다.

## 주요 특징
- **21세기의 핵심 스킬**: 과거의 프로그래밍 능력이나 검색 능력이 중요했던 것처럼, 이제는 AI를 자신의 "디지털 대리인"으로 얼마나 잘 활용하느냐가 생산성의 척도가 됩니다.
- **언어 기반 제어**: 영어(또는 자연어)로 논리적인 지침을 설계하고, 에이전트가 수행한 결과를 검증하며 피드백을 주는 과정이 핵심입니다.
- **도구 결합**: 에이전트가 단순한 대화를 넘어 파일 시스템, 웹 브라우저, 외부 API 등을 조작할 수 있도록 환경(예: [[MCP]])을 구축하는 능력이 포함됩니다.

## LLM-Wiki와의 연관성
