## Part 1. 핵심 문법 빈칸 채우기

### 1. 반응형 웹 (Responsive Web)
1. 화면 크기(해상도)에 따라 CSS 스타일을 다르게 적용하기 위해 사용하는 CSS 문법은 `[ 1 ]` 이다. (예: `@media (max-width: 600px) { ... }`)
2. 모바일 기기에서 뷰포트(Viewport)의 너비를 기기의 실제 너비에 맞추기 위해 HTML `<head>`에 삽입하는 메타 태그의 name 속성은 `[ 2 ]` 이다.

### 2. 자바스크립트 기본 문법 및 DOM
3. 자바스크립트에서 변수를 선언할 때, 재할당이 불가능한 상수를 선언하는 키워드는 `[ 3 ]` 이다.
4. HTML 문서 내에서 id 속성값이 "myBtn"인 요소를 찾기 위해 사용하는 순수 자바스크립트(Vanilla JS) 메서드는 `document.[ 4 ]("myBtn")` 이다.
5. DOM 트리에서 특정 요소의 내부 HTML 코드를 읽거나 변경할 때 사용하는 속성(Property)은 `[ 5 ]` 이다.

### 3. jQuery 라이브러리
6. jQuery를 사용할 때, id가 "box"인 요소를 선택하는 문법은 `[ 6 ]` 이다.
7. jQuery에서 특정 요소에 CSS 클래스 "active"를 추가하는 메서드는 `. [ 7 ]("active")` 이다.
8. 순수 자바스크립트의 `window.onload` 또는 `document.addEventListener("DOMContentLoaded", ...)`와 동일한 역할을 하는 jQuery의 문서 로드 완료 이벤트 처리 메서드는 `$(document).[ 8 ](function() { ... })` 이다.

## Part 2. 헷갈리기 쉬운 JS vs jQuery O/X 퀴즈 (객관식 대비)

1. jQuery에서 태그의 내부 텍스트만 가져올 때는 `.html()` 메서드를 사용해야 한다. `( O / X )`
2. 순수 자바스크립트 `document.querySelector(".item")`은 클래스가 item인 모든 요소를 배열로 반환한다. `( O / X )`
3. jQuery에서 요소를 화면에서 숨기려면 `.hide()` 메서드를 사용할 수 있다. `( O / X )`
4. 자바스크립트에서 `==` 연산자는 값과 데이터 타입(Type)을 모두 엄격하게 비교한다. `( O / X )`
