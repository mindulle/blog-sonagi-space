# Async/Await

`async/await` is a special syntax to work with promises in a more comfortable fashion.
We use `async` keyword to declare a async function that return a Promise, and the `await` keyword makes a function wait for a Promise.

# Async Await

Flutter's `async`/`await` pattern simplifies asynchronous programming by enabling code that appears synchronous. The `async` keyword designates a function as asynchronous, allowing non-blocking execution, while `await` pauses execution until an asynchronous operation completes, resulting in cleaner 
and more maintainable asynchronous code.

Visit the following resources to learn more:

- [@official@Asynchronous Programming: async, await](https://dart.dev/codelabs/async-await)
- [@official@Async widgets](https://docs.flutter.dev/development/ui/widgets/async)
