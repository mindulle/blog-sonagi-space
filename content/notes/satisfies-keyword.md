# Satisfies Keyword

The `satisfies` keyword in TypeScript is used to ensure that a value conforms to a specific type without explicitly declaring that type. This is particularly useful when you want to check that an object's structure matches a type definition but still allow TypeScript to infer a more specific type fo
r the object's properties. It validates the shape of the value against the specified type, and if valid it retains the initial type information.

Visit the following resources to learn more:

- [@official@satisfies Keyword](https://www.typescriptlang.org/docs/handbook/release-notes/typescript-4-9.html#the-satisfies-operator)
