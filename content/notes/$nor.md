# $nor

The `$nor` operator in MongoDB performs logical NOR operation, selecting documents that fail to match any of the specified query expressions. It's the inverse of $or and returns documents that don't satisfy any of the given conditions. `$nor` is useful for complex exclusion logic and finding documen
ts that don't match multiple alternative criteria.

Visit the following resources to learn more:

- [@official@\$nor](https://www.mongodb.com/docs/manual/reference/operator/query/nor/)
- [@official@Aggregation Operators](https://www.mongodb.com/docs/manual/reference/operator/aggregation/)
