## 0. 데이터 클리닝 및 EDA (Lecture 13-16)

### 🧹 데이터 클리닝 (Data Cleaning)
*   **Missing Values (결측치)**: `isnull()`, `dropna()`, `fillna()` 등을 활용. 
    *   *전략*: 삭제(Row/Column), 대체(평균, 중앙값, 최빈값 등 Imputation).
*   **Outliers (이상치)**: 데이터의 전반적인 패턴에서 크게 벗어난 값.
    *   *탐지*: Box Plot, Z-Score, IQR(Interquartile Range).
*   **Data Transformation**: 데이터의 스케일을 맞추거나(Scaling), 범주형 데이터를 수치형으로 변환(One-hot Encoding).

### 🔍 탐색적 데이터 분석 (EDA)
*   **목적**: 데이터의 구조와 특성을 파악하여 가설을 설정하고 모델링 방향을 결정.
*   **시각화 도구**: 
    *   `Matplotlib`, `Seaborn`.
    *   *종류*: Histogram(분포), Scatter Plot(상관관계), Heatmap(공분산/상관계수).

## 1. 머신러닝 기초 (Intro to ML - Lecture 17-18)

### 🧱 핵심 개념: 일반화(Generalization) vs 과적합(Overfitting)
*   **일반화**: 모델이 보지 못한(Unseen) 데이터에 대해 얼마나 잘 예측하는가.
*   **과적합(Overfitting)**: 모델이 학습 데이터의 노이즈(Noise)까지 학습하여, 학습 오차는 낮으나 테스트 오차는 높은 상태.
    *   *해결*: 정규화(Regularization), 데이터 증강, 모델 단순화.
*   **과소적합(Underfitting)**: 모델이 너무 단순하여 학습 데이터의 패턴조차 파악하지 못한 상태.

### 🧪 데이터 분할 전략
*   **Train Set**: 모델 파라미터 학습용.
*   **Validation Set**: 하이퍼파라미터 튜닝 및 모델 선택용.
*   **Test Set**: 최종 모델 성능 평가용 (Data Leakage 방지 필수).

## 2. 지도 학습 (Supervised Learning - Lecture 19-20)

### 📈 선형 회귀 (Linear Regression)
*   **원리**: $y = wx + b$ 형태의 직선을 찾아 실제값과 예측값의 차이(오차)를 최소화.
*   **손실 함수**: **MSE (Mean Squared Error)**. 오차를 제곱함으로써 큰 오차에 대해 더 강한 페널티를 부여.

### 📉 로지스틱 회귀 (Logistic Regression)
*   **특징**: 이름은 회귀지만 **'이진 분류(Binary Classification)'** 모델임.
*   **시그모이드 함수(Sigmoid)**: 출력값을 0과 1 사이의 확률값으로 변환. $S(z) = \frac{1}{1 + e^{-z}}$

### 👥 k-최근접 이웃 (k-NN)
*   **원리**: 새로운 데이터와 가장 가까운 $k$개의 이웃을 찾아 다수결로 결정.
*   **주의**: 거리 기반 알고리즘이므로 반드시 **Feature Scaling**이 필요함.

---

## 3. 의사결정 나무 (Decision Trees - Lecture 21-22)

### 🌳 분할 기준 (Splitting Criteria)
*   **엔트로피(Entropy)**: 데이터의 불순도(무질서도)를 측정. 0일 때 가장 순수.
*   **정보 획득(Information Gain)**: 부모 노드의 엔트로피 - 자식 노드 엔트로피의 가중 평균. 이 값이 **최대**가 되는 방향으로 분할.

---

## 4. 인공신경망 (Neural Networks - Lecture 21-22)

### 🕸️ 다층 퍼셉트론 (MLP) & 역전파 (Backpropagation)
*   **활성화 함수**: ReLU, Sigmoid 등. 모델에 비선형성을 추가.
*   **역전파**: 출력층의 오차를 입력층 방향으로 전달하며 **경사하강법(Gradient Descent)**을 통해 가중치를 업데이트함.

---

## 5. 군집화 (Clustering - 비지도 학습)

### 📍 k-Means 알고리즘
1.  $k$개의 중심점(Centroid) 무작위 선택 -> 2. 가까운 중심점에 할당 -> 3. 중심점 재계산 -> 4. 반복.
*   **Elbow Method**: SSE(Inertia)가 급격히 줄어들다가 완만해지는 지점을 최적의 $k$로 선택.

## 6. 데이터 윤리 (Data Ethics - Lecture 23)

### ⚖️ 주요 쟁점
*   **편향(Bias)**: 학습 데이터의 불균형이나 사회적 편견이 모델에 반영되는 현상.
*   **투명성(Transparency)**: 블랙박스 모델의 판단 근거를 설명하는 **XAI**의 중요성.
