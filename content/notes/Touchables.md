# Touchables

In React Native, `Touchable` components are used to handle user interactions like taps, long presses, and double-taps on the appropriate elements. Each of these components is from the `react-native` package, except `TouchableScale` which is from `react-native-touchable-scale`. They can be used inter
changeably depending on the type of interaction you want to provide. The main `props` used with these components are `onPress`, `onLongPress`, and some component-specific ones like `underlayColor` for
 `TouchableHighlight`.

Visit the following resources to learn more:

- [@official@Handling Touches](https://reactnative.dev/docs/handling-touches)
- [@official@TouchableOpacity](https://reactnative.dev/docs/touchableopacity)
