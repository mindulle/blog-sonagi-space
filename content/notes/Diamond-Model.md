# Diamond Model

The Diamond Model is a framework for understanding and analyzing cyber threat activity. It visualizes an intrusion event as a diamond shape with four core features: adversary, capability, infrastructure, and victim. Analyzing these elements and the relationships between them provides valuable insigh
ts into the nature of the attack, helping security professionals attribute, track, and defend against malicious campaigns.
