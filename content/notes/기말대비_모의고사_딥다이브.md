## 📝 Part 1. 실전 객관식 모의고사

**문제 1. 다음 중 JSON 데이터에 대한 설명으로 올바르지 않은 것은?**
① 데이터를 키(Key)와 값(Value)의 쌍으로 표현한다.
② 키(Key)와 문자열 값은 반드시 큰따옴표("")로 감싸야 한다.
③ 자바스크립트 객체 문법에서 파생되었으므로 오직 자바스크립트에서만 사용할 수 있다.
④ 네트워크를 통해 데이터를 주고받을 때 주로 문자열(String) 형태로 전송된다.

**문제 2. 자바스크립트 객체를 JSON 문자열로 변환할 때 사용하는 올바른 메서드는?**
```javascript
const myBook = { title: "코스모스", author: "칼 세이건" };
const converted = /* [ 빈칸 ] */;
```
① `JSON.parse(myBook)`
② `JSON.stringify(myBook)`
③ `myBook.toJSON()`
④ `Object.toString(myBook)`

**문제 3. 다음 코드의 실행 결과 (A)와 (B)에 들어갈 값으로 알맞은 것은?**
```javascript
const arr = [1, 2, 3];
const resultA = arr.map(x => x * 2);
const resultB = arr.filter(x => x > 1);
```
① (A): `[1, 2, 3]`, (B): `[2, 3]`
② (A): `[2, 4, 6]`, (B): `[2, 3]`
③ (A): `[2, 4, 6]`, (B): `[3]`
④ (A): `[1, 4, 9]`, (B): `[1, 2, 3]`

**문제 4. 다음은 외부 API에서 데이터를 가져오는 비동기 코드이다. 빈칸 (가)와 (나)에 들어갈 키워드로 알맞은 것은?**
```javascript
// (가) 함수 선언
(가) function fetchData() {
    try {
        // 데이터 요청 후 대기
        const response = (나) fetch('data.json');
        const data = (나) response.json();
        console.log(data);
    } catch (error) {
        console.error("에러 발생", error);
    }
}
```
① (가) async, (나) await
② (가) await, (나) async
③ (가) defer, (나) async
④ (가) async, (나) promise

**문제 5. 다음 중 배열 데이터를 HTML 문자열로 변환하여 화면에 렌더링하는 가장 효율적이고 올바른 코드는?**
① `container.innerHTML = arr.map(item => "<div>" + item + "</div>");`
② `container.innerText = arr.map(item => \`<div>${item}</div>\`).join('');`
③ `container.innerHTML = arr.map(item => \`<div>${item}</div>\`).join('');`
④ `container.innerHTML = arr.forEach(item => \`<div>${item}</div>\`);`

**문제 6. 순수 자바스크립트의 DOMContentLoaded 이벤트를 jQuery로 올바르게 변환한 것은?**
```javascript
document.addEventListener('DOMContentLoaded', () => {
    fetchData();
});
```
① `$(window).load(function() { fetchData(); });`
② `$(document).ready(function() { fetchData(); });`
③ `$('document').on('load', fetchData);`
④ `$(body).ready(fetchData);`

**문제 7. 다음 중 요소의 스타일을 변경하는 순수 자바스크립트와 jQuery 코드가 올바르게 짝지어진 것은?**
① JS: `el.style.display = 'none';` / jQuery: `$(el).hide();`
② JS: `el.className += ' active';` / jQuery: `$(el).css('active');`
③ JS: `el.innerHTML = 'Hello';` / jQuery: `$(el).text('Hello');`
④ JS: `el.style.color = 'red';` / jQuery: `$(el).color('red');`

## 💡 Part 2. 정답 및 딥 다이브 해설

* **문제 1 정답: ③**
  * **💡 딥 다이브**: JSON(JavaScript Object Notation)은 자바스크립트 객체 문법에서 파생되었지만, **언어에 독립적(Language Independent)**입니다. 파이썬, 자바 등 거의 모든 최신 프로그래밍 언어에서 기본적으로 지원하며 데이터를 교환하는 표준 포맷으로 사용됩니다. (작은따옴표 사용 불가 유의!)

* **문제 2 정답: ②**
  * **💡 딥 다이브**: 자바스크립트 객체를 네트워크로 전송하기 위해 문자열로 변환하는 것을 직렬화(Serialization)라고 하며 `JSON.stringify()`를 사용합니다. 반대로 수신된 문자열을 자바스크립트 객체로 변환(파싱)할 때는 `JSON.parse()`를 사용합니다.

* **문제 3 정답: ②**
  * **💡 딥 다이브**: `map`은 배열의 모든 요소를 순회하며 콜백 함수의 반환값으로 **새로운 배열을 생성**합니다 (1*2=2, 2*2=4, 3*2=6). `filter`는 콜백 함수의 조건식이 `true`인 요소만 남겨 **새로운 배열을 생성**합니다 (1보다 큰 2, 3만 남음).

* **문제 4 정답: ①**
  * **💡 딥 다이브**: 비동기 처리를 직관적으로 작성하기 위해 함수 선언부 앞에 `async`를 붙이고, 데이터를 기다려야 하는 비동기 작업(fetch 등) 앞에 `await`를 붙입니다. `await`는 반드시 `async` 함수 내부에서만 사용할 수 있습니다.

* **문제 5 정답: ③**
  * **💡 딥 다이브**: `map`의 결과는 배열이므로 이를 그대로 `innerHTML`에 넣으면 요소 사이에 쉼표(`,`)가 출력됩니다. 따라서 `.join('')`을 통해 배열을 하나의 문자열로 합쳐주어야 합니다. 템플릿 리터럴(` `)을 사용하면 HTML 태그 안에 변수를 깔끔하게 넣을 수 있습니다. (`innerText`를 사용하면 태그가 문자 그대로 화면에 노출되므로 주의!)

* **문제 6 정답: ②**
  * **💡 딥 다이브**: HTML 문서 로드가 완료되었을 때 코드를 실행하는 jQuery의 표준 문법은 `$(document).ready(function() { ... })` 입니다. 간소화된 형태인 `$(function() { ... })`로도 작성할 수 있습니다.

* **문제 7 정답: ①**
  * **💡 딥 다이브**: 요소를 숨길 때 jQuery는 `.hide()` 메서드를 제공하며 이는 `display: none` 과 같습니다. 클래스를 추가할 때는 `.addClass('active')`를 사용해야 하고, 내부 HTML을 조작할 때는 `.html()`을 사용합니다 (text는 태그를 해석하지 않음).
