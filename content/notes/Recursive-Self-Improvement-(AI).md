# Recursive Self-Improvement (AI)

AI 모델이 자기 자신(또는 다른 AI 모델)의 개발 과정(데이터 정제, 모델 학습, 코드 생성 등)에 직접 참여하여 지능을 점진적으로 향상시키는 개념입니다.

## 核心 (Core Insight)
- **Anthropic의 관점**: AI가 AI를 만드는 과정에서 발생할 수 있는 '정렬 문제(Alignment)'와 '안전성'을 어떻게 확보할 것인가가 핵심입니다.
- **주요 활용**: 합성 데이터 생성(Synthetic Data), 모델 아키텍처 자동 탐색 등.
