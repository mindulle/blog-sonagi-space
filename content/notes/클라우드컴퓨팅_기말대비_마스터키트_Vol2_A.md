# 📑 클라우드컴퓨팅 마스터키트 Vol.2-A (심화 알고리즘 분석)
**버전**: Vol. 2-A (심화 알고리즘 분석)
**시험 범위**: Lecture 09 ~ 20
**사용법**: 인쇄 후 직접 펜으로 빈칸을 채우거나 문제를 풀며 암기 상태를 점검하세요.

---

## Part 1. OpenStack 심화 서비스
1. OpenStack에서 사용자에게 웹 기반 대시보드 인터페이스를 제공하는 UI 서비스의 이름은 `[        ]`(Horizon)이다.
2. 가상 머신에 대한 오케스트레이션 및 템플릿 기반 자동 배포를 담당하는 서비스는 `[        ]`(Heat)이다.
3. 데이터베이스 서버를 서비스 형태로 제공하는 DBaaS(Database-as-a-Service) 컴포넌트는 `[        ]`(Trove)이다.
4. 클라우드 자원의 사용량을 모니터링하고 과금 정보를 수집하는 텔레메트리 서비스는 `[        ]`(Ceilometer)이다.

## Part 2. 컨테이너 및 마이그레이션 기술
5. 컨테이너 마이그레이션 중, 소스 노드에서 CPU 상태만 먼저 전송하고 타겟 노드에서 즉시 실행을 시작한 뒤, 필요한 메모리 페이지를 나중에 가져오는 방식을 `[        ]`(Post-copy)라 한다.
6. 도커 컨테이너 실행 시 호스트의 포트와 컨테이너의 포트를 연결하는 옵션은 `-p` 이며, 이를 `[        ]`(Port Forwarding)이라 한다.
7. 마이크로서비스 아키텍처(MSA)에서 서비스 간의 통신, 보안, 모니터링을 관리하기 위해 도입하는 인프라 계층을 `[        ]`(Service Mesh)라 한다.

## Part 3. 클라우드 보안 모델
8. "아무도 신뢰하지 않는다"는 원칙하에, 네트워크 내부라도 모든 접근에 대해 엄격한 인증을 요구하는 보안 모델은 `[        ]`(Zero Trust)이다.
9. 클라우드 자원에 접근하는 사용자의 행위를 분석하여 비정상적인 접근을 탐지하는 시스템은 `[        ]`(UEBA)이다.
10. 가상 네트워크 내에서 특정 인스턴스로 드나드는 트래픽을 제어하는 가상 방화벽 역할을 하는 것은 `[        ]`(Security Group)이다.
