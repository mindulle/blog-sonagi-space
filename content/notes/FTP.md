# FTP

File Transfer Protocol (FTP) is a standard network protocol used for transferring files between a client and a server over a TCP/IP network, such as the internet. It operates using a client-server model, where a client initiates a connection to an FTP server to upload, download, delete, or rename fi
les. FTP requires authentication, usually with a username and password, and establishes separate control and data connections for managing commands and transferring data, respectively.
