# 🌐 Visual Identity v4.0: The Blockchain Pivot

**결단**: 소나기의 메인 브랜드 정체성을 단정한 'Modern Architect'에서, 에이전트와 블록체인 생태계를 상징하는 **'Falling Matrix (사이버펑크 픽셀)'**로 전면 선회(Pivot)합니다.

## 1. 듀얼 트랙(Dual-Track) 디자인 전략
시각적 임팩트(Matrix)와 사용성(Clean UI)의 충돌을 막기 위해, 디자인 영역을 두 갈래로 나눕니다.

### 🔵 Track A: The Matrix (브랜드 & 임팩트 영역)
- **적용 대상:** 메인 로고(`brand-logo-v4-matrix.svg`), 스플래시 스크린, 404 에러 페이지, 에이전트 터미널 로딩 화면.
- **시각적 특징:** 5x7 픽셀 매트릭스, 네온 사이언(`#00ffcc`), 흘러내리는 빗방울/글리치 애니메이션.

### ⚪ Track B: The Clean (기능성 UI 영역)
- **적용 대상:** 본문 텍스트, 데이터 테이블, 기본 버튼, 내비게이션 바.
- **시각적 특징:** 사용자가 "어떤 부분은 단정하게 갈지 다시 정하자"고 합의한 영역. 앞서 제작한 **18종의 2px 코어 아이콘(Wild & Thin)**은 이 영역에 속하며, 가독성을 최우선으로 확보합니다.

## 2. 코어 컬러 업데이트
- 배경색은 순백색 대신 심연의 **Cyber Black (`#010609`)**을 기본값으로 상정합니다.
- 강조 컬러로 기존 Sonagi Blue (`#1991B9`) 위에 **Neon Cyan (`#00ffcc`)**을 덮어 픽셀의 발광(Glow) 효과를 극대화합니다.
