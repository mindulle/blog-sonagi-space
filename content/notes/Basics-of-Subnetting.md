# Subnetting Fundamentals

Subnetting is the practice of dividing a network into two or more smaller, logically isolated networks, called subnets. This is accomplished by manipulating the subnet mask, which defines the range of IP addresses that belong to a particular network. By carving up a larger network address space, you
 can improve network performance, security, and manageability by limiting broadcast domains and isolating traffic.
