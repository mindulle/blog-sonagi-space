## 상세 내용 (Merged from 20_Wiki/Develop/_commons/fruit/Convention/React/React.md)
In React, the naming convention for components is to use PascalCase, meaning the first letter of each word in the component's name should be capitalized. For example, `UserProfile`, `SidebarItem`, or `NavigationMenu`. This convention differentiates custom React components from regular HTML tags in J
SX, as React treats elements starting with a lowercase letter as DOM tags and those starting with a capital letter as custom components.
