# AI+사람 역할 분담 패턴

AI와 사람이 협업할 때 반복적으로 나타나는 역할 분담 구조를 정리합니다.
레시피를 만들 때 이 패턴 중 하나를 골라 뼈대로 씁니다.

---

## 핵심 원칙

AI에게 모든 것을 맡기지 않는다.
**"AI가 잘하는 것"**과 **"사람이 해야 하는 것"**을 명확히 나눈다.

| AI가 잘하는 것 | 사람이 해야 하는 것 |
|---|---|
| 반복적·수치적 작업 | 판단, 취향, 맥락 |
| 대량 생산, 초안 | 최종 결정, 편집 |
| 정확한 좌표/계산 | 창의적 방향 설정 |
| 파이프라인 자동화 | 품질 검수 |

---

## 패턴 목록

### Pattern A: 재료 준비 → 조립

AI가 재료를 만들고, 사람이 조립·완성한다.

```
AI: 도형 배치 / 에셋 생성 / 초안 대량 생산
         ↓
사람: 선택 / 조합 / 후가공 / 납품
```

**적합한 작업**: 로고 디자인, 일러스트, UI 에셋

---

### Pattern B: 구조 설계 → 내용 채우기

AI가 뼈대(구조)를 잡고, 사람이 내용을 채운다.

```
AI: 그리드 구성 / 레이아웃 / 정렬 수치 계산
         ↓
사람: 텍스트 입력 / 색상 결정 / 감성적 조정
```

**적합한 작업**: 편집 디자인, 프레젠테이션, 문서 작업

---

### Pattern C: 역산 → 레시피화

기존 에셋·결과물을 AI가 분석해 과정을 역산하고, 사람이 검증·정제한다.

```
사람: 레퍼런스 에셋 제공
         ↓
AI: 레이어 구조 분석 / 수치 역산 / 재현 절차 도출
         ↓
사람: 검증 / 수정 / 레시피로 확정
```

**적합한 작업**: 레퍼런스 분석, 기존 스타일 재현, 워크플로우 추출

---

### Pattern D: 병렬 생산 → 큐레이션

AI가 다수의 변형안을 생산하고, 사람이 고른다.

```
AI: 10~50개 변형안 생성 (색상, 구도, 스타일 등)
         ↓
사람: 큐레이션 / 조합 / 방향 결정
```

**적합한 작업**: 로고 시안, 컬러 팔레트, 아이디어 탐색

---

## 레시피와의 연결

각 레시피는 위 패턴 중 하나(또는 조합)를 명시합니다.
패턴이 명확할수록 나중에 다른 분야에 이식하기 쉽습니다.
