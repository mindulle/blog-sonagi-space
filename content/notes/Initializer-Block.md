# Initializer Block

An initializer block in Java is a block of code, enclosed in curly braces `{}` , that is executed when an instance of a class is created. It's used to initialize instance variables or perform setup tasks before the constructor is called. There are two types: instance initializer blocks, which run ev
ery time a new object is created, and static initializer blocks, which run only once when the class is first loaded.
