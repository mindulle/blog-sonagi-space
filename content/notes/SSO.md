# Single Sign-On (SSO)

Single Sign-On (SSO) lets a user access multiple applications and websites with just one set of credentials. Instead of needing to remember and enter different usernames and passwords for each service, a user authenticates once, and that authentication is then shared securely across various intercon
nected systems. This streamlines the login process and improves user experience.
