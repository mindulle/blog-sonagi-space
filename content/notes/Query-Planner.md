# Query Planner in PostgreSQL

The PostgreSQL query planner is an essential component of the system that's responsible for optimizing the execution of SQL queries. It finds the most efficient way to join tables, establish subquery relationships, and determine the order of operations based on available data, query structure, and t
he current PostgreSQL configuration settings.

Learn more from the following resources:

- [@official@Planner/Optimizer](https://www.postgresql.org/docs/current/planner-optimizer.html)
- [@official@Query Planning](https://www.postgresql.org/docs/current/runtime-config-query.html)
