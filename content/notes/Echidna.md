# Echidna

Echidna is a Haskell program designed for fuzzing/property-based testing of Ethereum smart contracts. It generates random sequences of calls to the contract and checks if certain user-defined properties hold true after each sequence. This helps developers uncover unexpected behavior and potential vu
lnerabilities in their smart contract code.

Visit the following resources to learn more:

- [@official@Echidna - Smart Contracts](https://secure-contracts.com/program-analysis/echidna/index.html)
- [@opensource@Echidna: A Fast Smart Contract Fuzzer](https://github.com/crytic/echidna/)
