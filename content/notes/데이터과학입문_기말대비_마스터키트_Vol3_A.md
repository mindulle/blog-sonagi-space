# 📑 데이터과학입문 마스터키트 Vol.3-A (실전 판단 및 윤리 점검)
**버전**: Vol. 3-A (실전 상황 및 데이터 윤리)
**시험 범위**: Lecture 13 ~ 23 (기말 전범위)

---

## Part 1. 실전 상황 판단 (O/X)
1. EDA 단계에서 이상치(Outlier)를 무조건 삭제하는 것이 모델의 정확도를 높이는 가장 올바른 방법이다. ( O / X )
2. 피어슨 상관계수가 0이라는 것은 두 변수 간에 아무런 연관성(비선형 포함)이 없음을 완벽히 보장한다. ( O / X )
3. 로지스틱 회귀의 결정 경계(Decision Boundary)는 기본적으로 직선(또는 초평면)의 형태를 띤다. ( O / X )
4. k-NN 알고리즘은 훈련 단계에서 별도의 수식(파라미터)을 학습하지 않고, 데이터 자체를 기억해두었다가 예측 시 계산을 수행하는 Lazy Learner이다. ( O / X )
5. 의사결정 나무는 데이터의 스케일(Scale) 단위에 큰 영향을 받으므로 반드시 정규화(Normalization) 과정이 필요하다. ( O / X )
6. k-Means 알고리즘은 초기 중심점(Centroid)을 무작위로 설정할 경우, 전역 최적해(Global Optimum)가 아닌 지역 최적해(Local Minima)에 빠질 위험이 있다. ( O / X )
7. 딥러닝에서 은닉층의 수가 많아질수록 모델은 복잡해지며 항상 과소적합(Underfitting)될 위험이 커진다. ( O / X )

## Part 2. 데이터 윤리 및 사회적 책임 (L23)
8. AI 모델이 과거의 불평등한 기록을 그대로 학습하여 특정 집단에게 불리한 결과를 내는 현상을 데이터 `[        ]`(Bias)라 한다.
9. 복잡한 AI 모델의 판단 이유를 인간이 이해할 수 있도록 제공하는 기술을 `[        ]`(XAI)이라 한다.
10. 개인정보 보호를 위해 데이터에서 식별 가능한 정보를 제거하거나 변환하는 조치는 `[        ]`이다.
11. 데이터 분석가는 분석 결과가 사회에 미칠 부정적 영향을 미리 고려해야 할 `[        ]`적 의무가 있다.
12. 데이터 수집 시 사용자의 명시적 동의(Consent) 없이 수집된 개인 데이터를 활용하는 것은 심각한 `[        ]`(Privacy) 침해 문제이다.
