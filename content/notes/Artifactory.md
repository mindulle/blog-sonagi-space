# Artifactory

Artifactory is a universal artifact repository manager developed by JFrog that supports virtually every major package format. It integrates deeply with CI/CD tools, provides fine-grained access controls, and includes features like artifact promotion across environments, dependency caching, and vulne
rability scanning.

Visit the following resources to learn more:

- [@official@Artifactory Website](https://jfrog.com/artifactory/)
- [@video@Key Features of JFrog Artifactory in 5 Minutes](https://www.youtube.com/watch?v=ZCNOtVj7bow)
