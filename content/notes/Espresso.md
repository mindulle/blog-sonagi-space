# Espresso

Espresso is a native testing framework for Android to write reliable UI tests. Google released the Espresso framework in October 2013 and, as of release version 2.0, Espresso is part of the Android Support Repository. One of the important features in Espresso is that it automatically synchronizes yo
ur test actions with the user interface of your application. The framework also ensures that your activity is started before the test runs. It can also force a test to wait until all observer backgrou
nd activities have finished, which is sometimes a problem with other testing frameworks.

Visit the following resources to learn more:

- [@official@Espresso Documentation](https://developer.android.com/training/testing/espresso#kotlin)
- [@opensource@Espresso Code Samples on GitHub](https://github.com/android/testing-samples)
- [@article@Espresso Testing Tutorial](https://www.tutorialspoint.com/espresso_testing/index.htm)
